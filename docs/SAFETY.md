# Market Safety Guidelines

TradeSelfCheck is designed for educational market-watch and journaling workflows.

## Allowed positioning

Use language such as:

- Market watch
- Setup tracker
- Educational view
- Risk-defined setup
- Confirmation level
- Invalidation level
- Target zone
- Journal review

## Avoid

Do not use language such as:

- Guaranteed profit
- Sure shot signal
- 100 percent accuracy
- Risk-free trade
- Buy now / sell now instruction
- Financial advice
- Auto-trading promise

## Content rule

Every generated market view should include:

1. Bias
2. Support
3. Resistance
4. Invalidation
5. Risk note
6. Disclaimer

## Disclaimer

This project is for learning and educational market-watch only. It is not financial advice, investment advice, or a recommendation system.
